## 从后往前贪心算法
```
    if nums == []:
 
        return False
 
    # 初始last是最后一个节点
 
    last = len(nums) - 1
 
    # 这里其实初始化为len(nums)-2,-1,-1 也是可以的，从倒数第二个节点开始判断是否可达倒数第一个节点，
 
    #但是从len(nums) - 1, -1, -1虽然冗余一次但是健壮性更好，譬如len(nums)=1时，1-2 = -1 不好
 
    for i in range(len(nums) - 1, -1, -1):
 
        if i + nums[i] >= last:
 
            last = i
 
    return last == 0
```
