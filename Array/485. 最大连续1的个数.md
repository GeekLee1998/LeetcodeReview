# 485. 最大连续1的个数                 
    
这道题其实和寻找数组最大连续和的思路是一样的，因为之前有做过，所以一下子就有思路并动手编码了。
需要注意的是判断的位置要写对，其他没有什么困难。 
  
# 优化  
看到别人有用乘法优化的（因为题目只有01），特地学习一下：  
```
        ans = 0
        c = 0
        for i in nums:
            c = i * c + i   # 简化了上述的判断语句，遇0置0，遇1叠加。
            if c > ans:
                ans = c
        return ans
```
